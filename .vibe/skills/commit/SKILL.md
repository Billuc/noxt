# Accept Skill

This skill commits the changes done.

The changes should be commited using Git.
The Git command to use is `git commit -m "<COMMIT_MESSAGE>"`.
The commit message should be generated by you.
In case there are no changes staged, but some changes have been made but not staged, stop and ask the user to use the `accept` skill beforehand.
If some changes have been changes and other changes have been made but not staged, stop and ask the user to use the `accept` skill beforehand.
